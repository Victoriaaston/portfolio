East Coast Guide

Home Page

I worked with three other developers during my General Assembly Software Engineering bootcamp to create an app called East Coast Guide. Our shared passion for food and travel inspired us to develop a platform where users can discover and share eats from the regions where we grew up.

We built the app using the Django framework and Python programming language, and used MongoDB to store data. We collaborated using Github for version control, and used CSS and Materialize for styling. We deployed the app on Heroku.

With the launch of the app, users can create and review listings in a directory, as well as update and delete the listings they have created. Currently, we are continuing to collaborate to expand the app by adding more locations, making the styling responsive, as well as including art shops, parks, events, and more to the platform.
